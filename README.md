


# 👶 Mia — Conectando e Auxiliando Mães na Maternidade

O **Mia** é um aplicativo web desenvolvido para ser um braço direito das mães durante a jornada da maternidade. O objetivo principal do projeto é oferecer suporte, organização e informações úteis para facilitar o dia a dia com os filhos, unindo um design acolhedor a uma interface intuitiva.

---

## 🚀 Funcionalidades Principais (Exemplos)

* **Acompanhamento de Rotina:** Registro de horários de amamentação, sono e fraldas.
* **Dicas de Maternidade:** Artigos rápidos e confiáveis sobre o desenvolvimento infantil.
* **Interface Acolhedora:** Cores e elementos visuais pensados para transmitir calma e segurança.
* **Design Responsivo:** Funciona perfeitamente em celulares, tablets e computadores.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as tecnologias fundamentais da web para garantir leveza e compatibilidade:

* **HTML5:** Estruturação semântica de todo o conteúdo do aplicativo.
* **CSS3:** Estilização personalizada, layout responsivo (Flexbox/Grid) e identidade visual.
* **JavaScript (ES6+):** Lógica do aplicativo, interatividade e manipulação dinâmica dos dados na tela.
